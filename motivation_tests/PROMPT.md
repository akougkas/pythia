# Case 001: Add Two Numbers

## Objective

Create a Python script that adds two numbers provided as command-line arguments
and prints the result. Include input validation and a `--help` flag.

## Constraints

- Must use `argparse`
- Must handle non-numeric input gracefully
- Must include a unit test file

## Expected Deliverables

- `add.py` — the CLI script
- `test_add.py` — unit tests
